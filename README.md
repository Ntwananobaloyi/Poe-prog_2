# Poe_2
Cybersecurity Awareness Chatbot
Overview

The Cybersecurity Awareness Chatbot is a desktop application developed in C# using Windows Presentation Foundation (WPF). The chatbot is designed to educate users about cybersecurity best practices, identify common user sentiments during security-related conversations, and provide guidance on staying safe online.

The application promotes cybersecurity awareness through interactive conversations, helping users understand threats such as phishing, malware, password attacks, social engineering, and unsafe online behavior.

Example Interaction

User:

I am curious about phishing emails.

Chatbot:

Good question. Phishing emails are fraudulent messages designed to trick users into revealing sensitive information. Always verify the sender and avoid clicking suspicious links.

User:

I think I clicked a suspicious link.

Chatbot:

It's good that you're paying attention. Disconnect from untrusted networks if necessary, run a security scan, and change passwords for affected accounts.
